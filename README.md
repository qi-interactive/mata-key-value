MATA Key-Value
==========================================

Generic Key -- Value store for MATA applications.

Installation
------------

- Add the module using composer: 

```json
"mata/mata-key-value": "~1.0.0"
```

-  Run migrations
```
php yii migrate/up --migrationPath=@vendor/mata/mata-key-value/migrations
```
